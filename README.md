# Academic Site Starter (Single‑File GitHub Pages)

This is a minimal, no‑Jekyll GitHub Pages template. It’s a single HTML page with a small CSS file.

## Quick Start

1. Create a new repository on GitHub named **`yourusername.github.io`** (replace with your GitHub username).
2. Upload these files to the root of that repository:
   - `index.html`
   - `assets/style.css`
   - `.nojekyll` (prevents Jekyll processing)
   - Optionally, add `cv.pdf` to link your CV.
3. Commit and push. Your site will be live at `https://yourusername.github.io` within a minute.
4. Edit `index.html` to add your bio, publications, and links.

## Optional
- Add a custom domain in **Settings → Pages → Custom domain**.
- Track visits with Google Analytics by adding the GA tag in `<head>`.

## Upgrading later
If you want a richer academic theme (blog, auto‑generated pubs), migrate to a Jekyll template like **academicpages**. The content you add here can be copied over.
